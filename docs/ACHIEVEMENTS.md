## Speedrun Checklist (Get 5 Achievements in One Session)

1. Create the repo on GitHub, push initial code
2. Open an issue "Fix: add .gitignore" → fix → close within 5 min → **Quickdraw ⚡**
3. Create a PR with a `Co-authored-by:` commit → merge without review → **YOLO 🎲** + **Pair Extraordinaire 🤝** + **Pull Shark 🦈**
4. Enable Discussions → post a Q&A → mark accepted → **Galaxy Brain 🧠**

That's 5 achievements in under 30 minutes.

## Tips

- Pair Extraordinaire works with ANY email in the `Co-authored-by:` trailer — the other person just needs a GitHub account linked to that email.
- Pull Shark counts merged PRs across ALL repos, not just this one.
- YOLO only needs one PR merged without review on any repo you can push to.
- Quickdraw timer starts when you OPEN the issue/PR, so have the fix ready before opening it.
- Galaxy Brain works on any public repo's Discussions, not just yours.
